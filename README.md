# Self-Service PPCall Integration Studio

A prototype demonstrating a self-service configuration portal for Pay-Per-Call publisher and buyer integrations.

## Features
- Interactive Wizard
- Raw JSON Editing
- Form-Based UI
- Test Console Simulation
- AI Configuration Assistant
- Real-time Freshness Tracking
- Mock Persistence Store (React Context + Local Storage)

## Getting Started

Install dependencies:
```bash
npm install
```

Start the local development server:
```bash
npm run dev
```

Run tests / linting:
```bash
npm run build
npm run lint
```
